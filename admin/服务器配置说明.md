## 当前配置

### 前端配置 (vite.config.js)

```javascript
server: {
  host: '0.0.0.0',  // 允许外部访问
  port: 3000,
  proxy: {
    '/api': {
      target: 'http://localhost:8000',  // 服务器内部访问后端（正确）
      changeOrigin: true
    }
  }
}
```

**说明**：`target: 'http://localhost:8000'` 是正确的，因为：
- 前端和后端在同一台服务器上
- 代理是在服务器内部进行的
- 外部用户访问 `http://42.194.134.223:3000` 时，前端会通过代理访问本地的 `localhost:8000`

确保 3000 端口已开放：

```bash

sudo ufw status
```

## 如果前端在本地运行

如果前端在本地电脑运行，需要修改 `vite.config.js`：

```javascript
proxy: {
  '/api': {
    target: 'http://42.194.134.223:8000',  // 改为服务器IP
    changeOrigin: true
  }
}
```

然后访问：`http://localhost:3000`
