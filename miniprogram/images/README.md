## 图标资源推荐

可以从以下平台下载免费图标：

1. **阿里巴巴矢量图标库** - https://www.iconfont.cn/
   - 搜索关键词：首页、扫码、订单、我的、回收
   - 下载 PNG 格式

2. **Font Awesome** - https://fontawesome.com/
   
3. **Feather Icons** - https://feathericons.com/

## 图片格式要求

- 格式：PNG（支持透明背景）
- TabBar图标：必须是静态图片，不支持网络图片
- 建议使用纯色图标，未选中为灰色，选中为绿色(#07C160)

## 临时方案

如果暂时没有图片资源，小程序会显示文字或使用Emoji作为替代。
部分页面使用了Emoji图标作为占位符。
